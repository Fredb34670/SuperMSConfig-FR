# Pile Technologique : SuperMSConfig (Francisation)

## 1. Langage & Core
- **Langage Principal :** C# (.NET Framework / .NET Core) - *Basé sur les extensions .exe et .dll.*
- **Format de Données :** JSON - *Pour les fichiers de configuration et plugins.*

## 2. Outils de Développement (Recommandés)
- **IDE :** Visual Studio ou JetBrains Rider.
- **Décompilateurs/Analyseurs :**
  - ilspycmd (Outil global .NET utilisé pour la décompilation automatique).
  - dnSpy / ILSpy (Pour l'analyse et la modification visuelle de l'IL).
  - Resource Hacker (Pour l'édition des ressources Win32).
- **Éditeur Hexadécimal :** HxD ou 010 Editor (Pour les patchs binaires directs si nécessaire).

## 3. Bibliothèques Détectées
- **Newtonsoft.Json :** Gestion de la sérialisation/désérialisation JSON.
- **Microsoft.Web.WebView2 :** Composant pour l'affichage de contenu web (Edge/Chromium).
- **PluginInterface :** Interface pour le système de plugins.

## 4. Gestion de Version
- **Git :** Pour le suivi des modifications (code décompilé et configurations).
